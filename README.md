# ML--Water-Leakage-Detection-System

## Overview

This project focuses on detecting and predicting water leakage using Machine Learning techniques.

The system performs data preprocessing, missing value handling, outlier detection, feature engineering, and applies multiple Machine Learning and Deep Learning algorithms to accurately identify leakage patterns.

The project also compares the performance of various models to determine the best algorithm for leakage prediction. 

---

## Features

* Data preprocessing and cleaning
* Missing value handling
* Outlier detection
* Feature encoding and scaling
* Multiple Machine Learning models
* Deep Learning using ANN
* Performance comparison using graphs and heatmaps

---

## Workflow

1. Load and preprocess the dataset
2. Handle missing values and outliers
3. Encode and scale features
4. Split dataset into training and testing sets
5. Train multiple ML models
6. Evaluate model performance using:

   * Accuracy
   * Precision
   * Recall
   * F1-Score
7. Compare results visually using graphs and heatmaps

---

## Results

The project compares multiple Machine Learning models for leakage detection and prediction.

Among all tested models, **AdaBoost achieved the best overall performance** based on evaluation metrics.
